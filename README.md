# Edvio AI Task Management Website
Edvio is a modern AI-powered task management platform designed for students and instructors. Built with the MERN stack (MongoDB, Express.js, React.js, Node.js) and Firebase for authentication, Edvio enables seamless course purchasing, scheduling, and communication through an integrated AI chatbot.



## Features:
🔹 For Students
User Registration & Login: Secure sign-up/sign-in using Firebase Authentication.

Buy Courses: Browse and purchase courses through Stripe integration.

AI Chatbot Assistance:

Get help choosing the right subject.

Generate and manage personalized class schedules.

User Dashboard:

View and update profile.

Track enrolled courses.

See transaction history.

Leave reviews for courses and instructors.

🔹 For Instructors
Student Interaction: scheduled classes.

Collaborate with Students: Assist students throughout the course lifecycle.

🔹 For Admins
Profile Management: Update personal admin details.

Course Management: Add and manage course content.

Student Oversight:

Monitor student activity.

Access all student assignments.

Scheduling Tools: Manage and track teaching schedules.

Analytics:

View insights into course performance and engagement.

AI Chatbot Monitoring: Oversee chatbot responses and efficiency.

Feedback & Resources:

Access student feedback.

Upload and manage course-related resources.

Logout Functionality.

## Tech stack
Frontend: React.js + Tailwind CSS

Backend: Node.js + Express.js

Database: MongoDB

Authentication: Firebase

Payments: Stripe

AI Integration: AI-powered chatbot for course help and scheduling

##🧪 Testing & Usage
Visit http://localhost:5173 for the frontend.

Use the platform to register users, buy courses, and interact with the AI chatbot.
